== Description ==

Change the site font to Japanese Google font. You can change the font type and weight as an option.

== Installation ==

1. Upload "Simple Google Fonts Japanese" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can set from the widget page.
